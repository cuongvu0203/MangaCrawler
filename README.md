## Getting Started

1. Mở project bằng Visual studio 2022 
2. Chọn Build để get nuget về
3. Run IIS để xem giao diện
4. Chọn mục "Leech truyện" --> Copy link từ sayhxxx.me về paste vào và thưởng thức

### Installation

1. Yêu cầu .NET 7 SDK
2. Visual studio 2022
3. MSSQL
